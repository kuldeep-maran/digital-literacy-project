## Project Overview

This repository contains my **Digital Literacy Portfolio Project** for the course **CSE0001 – Digital Literacy**.
The aim of this project is to understand how students can effectively use digital tools, maintain a professional online presence, communicate responsibly online, and stay safe from cyber threats.

Through this project, I explored multiple platforms, created digital resources, practiced professional communication, and learned about cybercrime awareness.

## Task Summary

### Task 1 – Digital Literacy Infographic

Created an infographic explaining the concept of digital literacy, useful tools for students, safe internet practices, and professional online presence using Canva.

### Task 2 – Student Digital Portfolio

Created and updated professional profiles on platforms such as GitHub and LinkedIn to build a professional digital identity.

### Task 3 – Coding & Collaboration Platforms

Explored coding practice platforms and completed a beginner challenge.
Also created a Google Form quiz to spread awareness about digital literacy among students.

### Task 4 – Email Etiquette

Drafted professional emails and created a checklist of responsible social media practices for students.

### Task 5 – Cybercrime Awareness

Researched a cybercrime case study and created a prevention checklist to help students stay safe online.
